# SaaS-tjänster för Webbexperten

Komplement till hemsida + SEO. Månadsvis återkommande intäkt.

## Auto-svar (AI Chatbot)
Automatiskt svar till kunder när företaget inte svarar.

- Chatt på hemsidan
- SMS-auto-svar vid missat samtal
- Messenger/WhatsApp-integration

## Bokningssystem
"Boka tid" utan att ringa.

- Byggs på Cal.com (open source)
- Synkar med Google Calendar
- Auto-påminnelse via SMS/mejl

## SMS-påminnelse
Minska no-shows med automatiska påminnelser.

- Twilio API
- Skickas 24h innan bokad tid

## Offert-verktyg
AI-genererade offerter från några få ord.

- Mallar per bransch
- Auto-beräkna pris baserat på bransch/prislista
- Skicka + följ upp via SMS

## Recensionshanterare
Samla in fler Google-recensioner automatiskt.

- Auto-skicka recensionslänk efter köp/besök
- AI-svar på alla recensioner
- Övervaka betyg

## Kundregister (CRM)
Enkelt CRM för småföretag.

- Namn, telefon, senaste besök, nästa bokning
- Auto-påminnelse: "Det var länge sedan — boka tid?"
- Exportera till Excel

## Faktura-påminnelse
Auto-påminnelse vid obetalda fakturor.

## Närvarokoll
Live-status på hemsidan om företaget är tillgängligt just nu.
