# Obsidian Vault — Hermes Agents

Detta vault är kopplat till Hermes agent-systemet på VPS:en. Agenterna (Scout, Scribe, Reach, Dev) läser och skriver anteckningar här.

## Struktur

```
Obsidian/Hermes/
├── inbox/          — nya anteckningar
├── research/       — Scout's research
├── writing/        — Scribe's texter
├── marketing/      — Reach's kampanjer
└── dev/            — Dev's tekniska anteckningar
```
